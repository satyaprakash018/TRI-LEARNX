# 📚 TRI-LEARNX (Study Portal)

An advanced web-based Study Portal designed to provide students with a centralized platform for accessing study materials, videos, question papers, and personalized learning features.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 📊 Personalized Dashboard
- 📁 Upload & Access Study Materials
- 🎥 Video Learning Section
- 📄 Previous Year Question Papers
- ⭐ Bookmark Important Resources
- 👤 User Profile Management
- 🛠️ Admin Panel for Content Management

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, Bootstrap
- Backend: Python (Flask)
- Database: MySQL / MongoDB
- Others: JavaScript

## ⚙️ Installation & Setup

1. Clone the repository
git clone https://github.com/satyaprakash018/TRI-LEARNX.git

2. Go to project folder
cd TRI-LEARNX

3. Create virtual environment
python -m venv venv

4. Activate environment
venv\Scripts\activate

5. Install dependencies
pip install -r requirements.txt

6. Run the project
python app.py

---

## 🔐 Environment Variables

Create a .env file and add:

SECRET_KEY=your_secret_key  
DATABASE_URL=your_database_url
